PROJECT 2

Project Title:

BMI Calculator


Description:

Developed a straightforward BMI (Body Mass Index) Calculator. This web application is designed to help users quickly assess their BMI, a key indicator of overall health.


Technologies Used:

The project is implemented using HTML, CSS, and JavaScript, ensuring a seamless and interactive user experience.


Project Images:


  • Home Page


Project Outcome:

The primary goal of this project is to offer a user-friendly BMI calculator that is easy to navigate. Users can input their weight and height, and the calculator provides an instant BMI calculation, along with an interpretation of the result. The project aims to promote health awareness and encourage users to maintain a healthy lifestyle.


Challenges Faced:

While developing this project, I encountered a few challenges, such as handling user input validation to ensure accurate calculations. Additionally, achieving a responsive design to accommodate various devices and screen sizes required careful consideration. Overcoming these challenges contributed to the overall learning experience and improved my problem-solving skills.
